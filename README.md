# FlightManagement_Repo

Repository Summary
A repository for the Airline Flight Management Database including the command-line interface application, using sqlite3 to connect and interact with the SQLite database. There is also a folder named queries.sql containing the sql queries. 

Setup Instructions
To clone the repository, open the terminal window and in Bash enter git clone https://github.com/jc-dev-create/flightmanagement_Repo.git

Extensions Required
Install SQLite3 and SQLite Viewer extensions

Running Instructions
To run the flight management menu interface, navigate to the project folder by typing the below into the terminal:
cd /workspaces/FlightManagement_Repo

Then to run the CLI interface, input: 
python flight_management_CLI/main.py
